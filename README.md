## 🚀 Getting Started

### Prerequisites
- **Node.js** (v16 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/memoryticket.git
   cd memoryticket
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   
   Create a `.env.local` file in the root directory and add your API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   
   Navigate to `http://localhost:3000` to see MemoryTicket in action!
